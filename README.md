# Kirati Heart

**Kirati Heart** is a digital sanctuary dedicated to preserving, celebrating, and sharing the rich cultural heritage of the Kirati people. This platform serves as a bridge between the ancient traditions of the eastern Himalayas and the modern world, ensuring that the legacy of the Kirati ancestors continues to thrive.

## 🌟 Key Features

-   **Interactive Timeline**: A visual journey through the history of the Kirati civilization, from ancient origins to the present day.
-   **Cultural Archive**: A living library containing traditional recipes, folk songs, and historical documentation.
-   **Oral Traditions**: A collection of myths, legends, and stories passed down through generations.
-   **Community Hub**: A space for events, language preservation, and connecting with the global Kirati community.

## 🛠️ Technology Stack

-   **Frontend**: React, TypeScript, Tailwind CSS, Vite
-   **Animations**: Framer Motion, ScrollReveal
-   **Icons**: Lucide React
