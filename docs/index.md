# Welcome to Ikirati

**Ikirati** is a Cultural Heritage Platform designed to preserve and showcase the rich heritage of the Ikirati community. This platform allows for the collection, meaningful organization, and accessible presentation of cultural stories, artifacts, and traditions.

## Overview

The project consists of two main components:

- **Backend**: A Node.js/Express server that manages data persistence, authentication, and API endpoints.
- **Frontend**: A React application (built with Vite) that provides an interactive user interface for exploring and managing cultural content.

## Quick Start

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd ikirati
   ```

2. Install dependencies for all packages:
   ```bash
   npm run install:all
   ```

3. Start the development servers:
   ```bash
   npm run dev
   ```

This will launch both the backend and frontend in development mode.
- Frontend: [http://localhost:5173](http://localhost:5173)
- Backend: [http://localhost:3000](http://localhost:3000)
