# Frontend Documentation

The frontend is a single-page application built with React, TypeScript, and Vite. It serves as the user-facing interface for the Ikirati platform.

## Setup

Navigate to the frontend directory:
```bash
cd frontend
```

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

## Technology Stack

- **Framework**: React
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Route components (HomePage, CraftsPage, EldersPage, etc.)
- `src/data`: Static data or data types
- `src/assets`: Static assets like images and fonts

## Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` directory.
